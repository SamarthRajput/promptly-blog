To start the project locally

1. Starting the Drizzle ORM locally 
    - Run commands locally
        1. ` npm run db:migrate `
            to migrate the database
        2. ` npm run db:generate `
            to generate or to apply migrations
        3. ` npm run drizzle-kit studio `
            to see the studio or the database schema on website 

2. Starting the project
    1. Set all the env using the .env.sample file
    2. install all the dependencies
        ` npm i `
    3. run the project
        ` npm run dev `
